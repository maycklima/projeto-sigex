# Descrição do projeto

Desenvolvimento de aplicação java utilizando JDBC

#Funcionalidades

#Usuário
- Cadastrar usuário
- Editar usuário

#Autenticação
- Login
- Logout
- Recuperar senha

#Aluno
- Cadastrar aluno
- Editar aluno
- Deletar aluno
- Ver todos os eventos que o aluno participa

#Evento
- Cadastrar evento
- Adiciona aluno ao evento
- Editar evento
- Cancelar evento
- Ativar evento
- Excluir evento
- Registro de ocorrencias dentro do evento

#Certificados
- Gerar certificado do evento para o aluno
